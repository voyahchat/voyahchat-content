1. [Включить USB Debugging](usb-debugging.md)

2. Подключить [кабель](cable.md) в порт USB Type-A внизу между водителем и пассажиром. Если в машине два USB Type-A — в левый, ближе к водителю

3. Выполнить установку приложения на [Windows](#windows) или [Mac](#mac)

4. Запустить, войти в `Настройки`, нажав три точки в правом верхнем углу
  * Выбрать `2` колонки вместо `6`
  * В списке радиостанций нажать на три точки у радиостанции, добавить её в избранное
  * В правом верхнем углу включить сердечко
  * В правом верхнем углу включить рядом с сердечком показ плиткой
  * Выключить плитку, выключить сердечко, настроить радиостанции, включить сердечко/плитку
  * Развернуть радиостанцию на весь экран, нажать на 32K, чтобы стало 48K

# Windows

1. Скачать [voyahchat-fmplay.zip](https://github.com/voyahchat/voyahchat-install/raw/refs/heads/main/voyahchat-fmplay.zip)

2. Разархивировать `voyahchat-fmplay.zip` в любую директорию, нажав на файл правой кнопкой в Проводнике и выбрав `Извлечь всё`

3. Войти в директорию `voyahchat-fmplay`

4. Запустить пакетный файл Windows `voyahchat-fmplay.bat`

# Mac

Запустить `Терминал` (стандартное приложение на Маке), выполнить последовательность команд. Копировать и вставлять в терминал по одной строке, после каждой команды нажимать `Enter`:
  * Перейти в директорию `Downloads`
```
cd ~/Downloads
```
  * Скачать voyahchat-fmplay-mac.zip
```
curl -O https://github.com/voyahchat/voyahchat-install/raw/refs/heads/main/voyahchat-fmplay-mac.zip
 ```
  * Разархивировать
```
unzip -o voyahchat-fmplay-mac.zip
```
  * Выполнить скрипт установки
```
sh ./voyahchat-fmplay.sh
```
